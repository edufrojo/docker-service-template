## Despliegue

Descargar repositorio:

```sh
git clone https://github.com/edufrojo/docker-service-template.git
```

Instalar dependencias:

```sh
npm install
```

Arrancar servidor:

```sh
npm run start
```

Acceder a través del navegador a `localhost:8080`
